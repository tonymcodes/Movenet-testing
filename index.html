How it works: MoveNet returns 17 body keypoints per frame via TensorFlow.js. For the chosen exercise this prototype picks 3 keypoints (e.g. hip → knee → ankle for squats), computes the joint angle with atan2, then a two-state machine (UP ⇄ DOWN) counts a rep each time the angle crosses both thresholds. The rep counter and stats are drawn onto the canvas, so screenshots and clips include the moving counts and overlay. Everything runs on-device — nothing is uploaded.
